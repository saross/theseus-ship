# Grimoire

A collection of reusable approaches and workflows for human-AI collaborative tasks.

Each entry documents an emergent pattern that proved effective, with:

- **Context**: When to use this approach
- **Workflow**: Step-by-step process
- **Model prompt**: Template for invoking the approach
- **Variations**: Adaptations for different contexts
